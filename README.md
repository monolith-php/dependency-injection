# Monolith: Dependency Injection

### Installation

In composer.json
```json
{
  "require": {
    "monolith/dependency-injection": "dev-master",
    "monolith/collections": "dev-master"
  },
  "repositories": [
    {
      "type": "vcs",
      "url":  "git@github.com:mitchellvanw/monolith-dependency-injection.git"
    }
  ]
}
```
